# linkedin-OSINT-
An interactive OSINT awareness simulation built with React. 

Presents a fictional LinkedIn profile (James R. Holloway, IT Support at MediCore Solutions) 
with 8 embedded vulnerabilities each clickable element reveals the real-world attack vector 
it exposes: spear phishing, credential stuffing, infrastructure reconnaissance, vishing windows, 
and more. Includes a severity scoring system (LOW → CRITICAL), a live findings log, and a 
full OSINT Intelligence Report view with executive summary and mitigations.

Built as a security awareness training tool to demonstrate how routine social media activity 
leaks an exploitable attack surface no hacking required.

Tech: Vanilla React (CDN) · No backend · Single HTML file · Zero dependencies
